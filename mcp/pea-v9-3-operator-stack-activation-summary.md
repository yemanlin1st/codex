# PEA V9.3 Operator Stack Activation Summary

Date: 2026-06-16
Reference: PEA Account Kernel V9.3

## Purpose

Use five MCP capability classes as a governed operator stack for creators, marketers, developers and business owners.

## Stack

1. Perplexity MCP: research and source-backed benchmarking.
2. Playwright MCP: controlled browser QA and interface testing.
3. Firecrawl MCP: structured public website analysis.
4. Glif MCP: creative draft generation for visual concepts.
5. Chrome MCP: supervised browser context and interaction.

## Operating policy

- Start read-only where possible.
- Use sandbox mode for browser testing.
- Keep creative outputs as drafts until reviewed.
- Respect public/private/confidential separation.
- Keep human approval for sensitive actions.
- Keep citations for research outputs.
- Apply anti-bias and zero-noise gates.

## Status

This stack upgrades the account to PEA V9.3 operator mode.

## Rule

Use MCP only when it improves continuity, research, QA, creativity, productivity or business execution.
